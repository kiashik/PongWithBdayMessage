# PongWithBdayMessage
This is a little project I made for a friend. It wish my friend a happy birthday and allow them to play the classic Pong game or use the screen as a digital image frame.

**Hardware**
1. Arduino Uno/Nano x1,
2. push buttons x4,
3. LCD screen,
4. Some resistors ~ 10 kohms.

**Schematic**
Follow the basic push button wiring with a 10 kohm resistor. will update with a schematic when i have time
